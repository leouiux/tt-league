용문리그·토너먼트
운영시스템 시작하기

리그전 결과 관리부터 토너먼트 대진표, 경품 추첨까지 — 처음 사용하시는 분도 쉽게 따라할 수 있는 단계별 안내입니다.

① 대회 만들기
② 명단 등록
③ 리그전
④ 토너먼트
⑤ 저장
🏓

이 시스템은 어떻게 사용하나요?

전체 운영 흐름을 먼저 파악하면 훨씬 쉬워집니다

🗺️ 전체 운영 흐름
1

대회 기본 정보 설정

날짜, 대회명, 조 수, 조별 인원, 탈락 인원, 경기 방식(2선승/3선승)을 입력합니다.

2

선수 명단 등록

각 조에 선수를 입력하거나 한번에 등록으로 단체카톡 명단을 붙여넣기합니다. 자동으로 조에 균등 배분됩니다.

3

리그전(조별 예선) 진행

각 조의 결과 매트릭스에 경기 결과를 입력하면 순위가 자동으로 계산됩니다. 드래그&드롭으로 순위 수동 조정도 가능합니다.

4

토너먼트(단판 결선) 진행

리그전 순위를 바탕으로 단판 토너먼트 대진표가 자동 생성됩니다. 같은 조끼리는 1회전에서 만나지 않도록 자동 배치됩니다.

5

저장 및 기록 보관

💾 저장 버튼으로 언제든지 저장하고, 📋 기록 버튼으로 과거 대회를 불러올 수 있습니다.

📊
자동 순위 계산
승점 → 승자승 → 세트득실 → 상위부 순으로 자동 계산됩니다
🏆
토너먼트 자동 배치
리그 순위 기반 시드 배정, 같은 조 1회전 충돌 자동 회피
🎁
경품 추첨 연동
리그전 명단이 경품 추첨 페이지로 가나다순 자동 전달
🖨️
인쇄 기능
전체 명단, 조별 대진표, 토너먼트 경기 카드 인쇄 지원

전체 경품 추첨

리그전 명단이 자동으로 추첨 페이지에 전달됩니다

🎲 사용 방법
1

🎁 전체경품추첨 버튼 클릭

리그전 배너 또는 첫 화면에서 클릭합니다. 리그전 진행 중이라면 현재 명단이 자동 전달됩니다.

2

경품 등록

경품 이름과 개수를 입력합니다. (예: 치킨 2개, 아메리카노 5개)

3

🎲 경품 랜덤 추첨

버튼 클릭 한번으로 자동 랜덤 추첨이 완료됩니다. 결과는 인쇄할 수 있습니다.

💡
리그전 명단은 가나다 순으로 자동 정렬되어 전달됩니다. 명단이 제대로 오지 않으면 팝업을 닫고 다시 버튼을 클릭해 보세요.
⚖️

동률 승자 계산

복잡한 동률 상황을 빠르게 해결합니다

3명 이상이 동률일 때 ⚖️ 동률간 승자 계산하기 도구를 사용하세요. 동률 인원 수 설정 → 결과 입력 → 순위 자동 계산 순으로 진행합니다.

💡 이름 입력 형식
이름+부수 형태: 홍길동2, 김철수4
⚠️

주의 사항

반드시 읽어주세요

⚠️ 꼭 지켜주세요

자동 저장이 없습니다. 결과를 입력한 후에는 반드시 💾 저장 버튼을 눌러주세요. 브라우저를 닫거나 새로고침하면 저장하지 않은 내용은 사라집니다.
중복 이름은 오류를 유발합니다. 명단에 같은 이름이 두 번 들어가면 순위 계산과 토너먼트 배정에 오류가 발생합니다. 빨간색 경고 표시가 나타나면 즉시 수정하세요.
토너먼트로 전환 후 리그전 결과를 수정하면 토너먼트 대진표가 초기화됩니다. 리그전 결과가 모두 확정된 후 토너먼트를 진행하세요.
기록 삭제는 복구가 불가능합니다. 삭제 전 📤 내보내기로 JSON 백업을 해두는 것을 권장합니다.
브라우저 여러 탭에서 동시에 사용하지 마세요. 동시 접속 시 데이터 충돌이 발생할 수 있습니다.

자주 묻는 질문

항목을 클릭하면 답변을 확인할 수 있습니다

리그전 도중 선수를 교체하거나 빠지면 어떻게 하나요?
▼
배너의 ✏️ 명단 한번에 수정을 클릭하고, 왼쪽(취소 명단)에 빠지는 선수, 오른쪽(새 명단)에 대체 선수를 입력하세요. 새 명단이 적으면 나머지 자리는 공석(최하위)으로 처리됩니다.
토너먼트에서 같은 조끼리 1회전에서 만나는 것 같아요
▼
시스템이 자동으로 같은 조 충돌을 최대 10회 재시도하여 해소합니다. 조 수가 적거나 진출 인원 구조상 불가피한 경우에만 같은 조 대전이 발생하며, 이때는 안내 메시지가 표시됩니다.
부전승이 엉뚱한 선수에게 배정됐어요
▼
부전승은 리그전 순위 상위 선수(각 조 1위부터)에게 자동 배정됩니다. 배정이 이상해 보인다면 토너먼트를 재생성해 보세요.
과거 대회 데이터를 다른 기기에서 보려면 어떻게 하나요?
▼
📤 내보내기로 JSON 파일을 저장한 뒤, 다른 기기에서 📂 가져오기로 불러오면 됩니다. 또는 데이터가 서버에 저장되어 있다면 같은 URL로 접속하면 자동으로 불러와집니다.
동률 선수가 많아서 자동 순위 계산이 안 돼요
▼
모든 기준(승자승, 세트득실, 등급)이 동일하면 연장자 우선으로 수동 확인이 필요합니다. 순위표에 노란색 배경이 표시됩니다. 드래그&드롭으로 순위를 직접 지정하세요.
경품 추첨 페이지에 명단이 제대로 전달되지 않아요
▼
경품 추첨 창을 닫고 다시 🎁 전체경품추첨 버튼을 클릭해 보세요. 팝업 차단 설정이 되어있다면 브라우저에서 팝업을 허용해 주세요. 그래도 안 되면 경품 추첨 페이지의 📋 명단 한번에 입력으로 직접 입력할 수 있습니다.
